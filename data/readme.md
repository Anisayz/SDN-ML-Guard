Large files not tracked by Git

Some files are not included in this repository because they are too large for GitHub. These are all auto-generated by running the pipeline so you don't need them to get started.

The excluded files are train.pkl, test.pkl, test_raw.pkl and benign_train.pkl which all go in data/processed/ and are generated by preprocess.py.
To regenerate everything, drop the CIC-IDS2018 CSVs into data/raw/ then run preprocess.py, then train.py, then evaluate.py in that order. That will recreate all the missing files.
